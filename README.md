# Humanoid Atlas

A bilingual interactive textbook for learning humanoid robots from mechanical anatomy to robot foundation models, VLA systems, control, safety, and data engines.

The site is static HTML, CSS, and JavaScript. It includes:

- A clickable humanoid robot overview
- English and Chinese language switching
- Nested lessons for major robot systems
- An advanced systems map covering LLM/VLM/VLA concepts, robot foundation models, information flow, safety gating, deployment reliability, and robot data engines
- Mobile-friendly layouts

## Local Preview

```bash
python3 -m http.server 8123
```

Then open `http://127.0.0.1:8123/`.

## Deploy

This folder can be deployed directly to Vercel as a static site.
